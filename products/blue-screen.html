About Blue Screen

When we are going to leave from the comuter or just want to make jokes, you can try this tool.

What Is It?

We usually see blue screen when we are using Windows and the system have to deal with some problems. At that time, the system is failed and we can't do anything, even screenshots. When we want to keep them, but you don't just want to read log or search stopcode to record them, this kind of tool can help you. It can not only generate normal photos but also put emojis in it to make jokes.

How It Works?

PIL.Pillow is a very strong photo-dealing library. We can generate photos look like Windows blue screen. We set up a new blue photo, and put the texts into the photo. We use Windows font to sopport Chinese.

How Can It Develop?

Even though the core generator has been done, but it's so hard to generate photos yourself.
1. We can generate a UI, or start with a console UI.
2. We can try to change background.
3. We can add "Preview" to the tool.
