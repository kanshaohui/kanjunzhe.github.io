About Event Recorder

When you are going to make a report for the students' parents about theirschool life but you can not remember anything happened in the class, or you want to criticize some student but you can not find any examples to use, you can try to use this kind of tool.

What Is It?

Event Recorder is a kind of tool that you can record anything happened at school and give it a score to judge if the event is good or not. No one can change the record file.

How It Works?

It was developed by Python. First create a file to record events. Then design an interface to input the requests and write the file. At last, use pyinstaller to package it into a executable file.

How Can It Develop?

Now, Event Recorder is just OK for schools to use, because most computers in school use Windows.
But, for Linux and MacOS users, we need to develop more.
1. Sopport more platforms like linux and MacOS.
2. Develop a UI.
