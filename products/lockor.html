About Lockor

When we are leaving the opening comuter for a while for a cup of coffee but afraid that someone watch your chat record, or worring the students will play with the comuter after class, you may be try this kind of tool.

What Is It?

We professional programmers have created a mouse-and-keyboard locker for you to use. When you start to use this tool, you will find it powerful. With it, you can makr sure that your computer is pretty safe. You can go out shopping and make the computer open to download something, or run some computationally intensive programs like AI model and no one can unlock it. Even reboot can't!

How It Works?

It was generated with c++. In c++, Microsoft company provided a great head file only run on Windows called Windows.h. It can hook the mouse and the keyboard inputs. When we use the function to hook the inputs, the only thing to do is change the value. We return 1 to the system and the system will know that there's no inputs from both the mouse and the keyboard. When you move the mouse or press down a button on the kayboard, the input event will be changed into "no events from te mouse or the kayboard". Now, you can use it rightly.
But when someone reboot the computer, the program will be stopped and all your data will also be stolen. Then we can edit the reg. We put the process to the reg to make it start automatically when the system started. Then your data is proably safe.

How Can I Unlock It?

Because the program is in the reg, you can not unlock is with several reboot. The only way to do is below:
1. Reboot the computer by pressing down the power button.
2. Don't login and reboot again with pressing down button "Shift".
3. After your PC restarts to the Choose an option screen, select Troubleshoot > Advanced options > Startup Settings > Restart.
4. After your PC restarts, you'll see a list of options. Select F4 to start your PC in safe mode.
5. Login the computer and press down Windows logo key + R.
6. Type regedit in the Open box and then select OK.
7. Open \HKEY_CURRENT_USER\Software\Microsoft\Windows\ CurrentVersion\Run and find Locker and delete it.
