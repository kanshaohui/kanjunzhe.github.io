Detail (The idea of the passage)

The idea of the passage is to show the complexity of the log file.
The log file is a file that contains the information of the computer system.


OK, let me think how should I tell you the truth.

Translation is not a simple work. You not only need to remember all the meaning of the words, find the best way to translate to make it "beautiful", but also the names. From Chinese to English, we usually just use pinyin, but it's not really the best way to translate. It's good to translate the Chinese name into an available English word. The word don't need to be very long, but it should be similar to the Chinese name in speaking. This can make the translation more fun.

A great example is Rogar. Rogar is the name of a Chinese character. You can see that this kinf of name is both similar to the original name and "beautiful". But, you know, the translation is not always like y = x, it usually have many different ways to translate. Recently, F has translated Rogar into Yogurt. Therefore, we don't need to be so serious about the translation, so you can find a great translation in my passage.

But pronounce is not the only way for you to translate the names. The first letter of pinyin of all the characters is also a good way to translate the names. YYY is also a good example. YYY is the name of a Chinese character and his name is Y*n Yu***n. In this passage, you can also find some good examples.

Take the passage seriously and you can find something more than you thought.
